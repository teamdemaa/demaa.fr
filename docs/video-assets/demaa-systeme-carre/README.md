## Duree recommandee

- total: `22s` a `27s`
- format: `1080x1080`
- transitions: fondu doux ou coupe simple

## Script de montage

### Plan 1

- fichier: `00-ouverture-message.png`
- duree: `3s`
- texte affiche: deja inclus dans l'image
- mouvement: leger zoom in `102% -> 106%`

### Plan 2

- fichier: `01-hero-systemes.png`
- duree: `3s`
- texte optionnel: `Trouvez les bons systemes`
- mouvement: leger zoom in

### Plan 3

- fichier: `02-recherche-btp.png`
- duree: `3s`
- texte optionnel: `Choisissez votre activite`
- mouvement: aucun ou tres leger zoom

### Plan 4

- fichier: `03-page-batiment.png`
- duree: `4s`
- texte optionnel: `Devis, chantiers, equipes, marge`
- mouvement: leger zoom centre sur le titre `B√¢timent`

### Plan 5

- fichier: `04-operations-ouvertes.png`
- duree: `5s`
- texte optionnel: `Un systeme concret, metier par metier`
- mouvement: pan vertical tres leger

### Plan 6

- fichier: `05-fin-message.png`
- duree: `4s`
- texte affiche: deja inclus dans l'image
- mouvement: aucun

## Musique

Choisir une piste:

- instrumentale
- douce mais rythmique
- sans voix
- pas trop corporate agressif
- ambiance: clarte, maitrise, respiration

## Important

Le pack contient les visuels de base, mais pas un export video `mp4` final ni la musique integree.
Dans cet environnement, je n'ai pas d'outil d'export video/audio pret a sortir directement un montage final avec musique.
